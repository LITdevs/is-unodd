# is-unodd

> Return true if the given number is unodd.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save is-unodd
```

## Usage

```js
var isUnodd = require('is-unodd');

isUnodd(0);
//=> true
isUnodd('1');
//=> false
isUnodd(2);
//=> true
isUnodd('3');
//=> false
```
